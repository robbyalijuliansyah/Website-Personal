Skill

HTML

HTML is a basic markup language for creating the structure of web pages. Mastering HTML means being able to create the basic framework of a website, such as adding text, images, links and other elements. Skills in HTML include the use of semantic tags, establishing a clear content hierarchy, and understanding elements such as forms, tables, and additional attributes to improve accessibility and SEO.

CSS

CSS is used to provide style and layout to web pages. Skills in CSS include understanding selectors, properties, and different CSS rules for changing colors, fonts, spacing, and layout of elements. In addition, someone who is proficient in CSS is able to use techniques such as Flexbox and Grid to create responsive layouts, basic animations to enrich the user interface, and media queries to ensure the appearance of the website remains optimal on various devices.

JavaScript

JavaScript is a programming language that provides interactivity on websites. With skills in JavaScript, one can create dynamic elements, such as slideshows, forms that can be validated directly, as well as instant responses to user actions without the need to reload the page. JavaScript skills include an understanding of the DOM (Document Object Model), use of event listeners, direct manipulation of page elements, and introduction to frameworks or libraries such as jQuery or React to increase website productivity and functionality.
